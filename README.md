# 【Java计算机毕业设计分享】桥牌计分系统

## 前言

此项目为桥牌计分系统的毕业设计，采用Java语言开发，结合Spring Boot框架、前端JS、Vue以及CSS3等技术实现。在完成本项目过程中，我们注重实战，将理论与实践相结合，力求为用户提供一个功能完善、易于使用的桥牌计分系统。

## 内容介绍

桥牌计分系统主要用于帮助桥牌爱好者进行比赛计分，支持多种计分规则，界面友好，操作简单。本项目包括前端展示、后端处理以及数据库管理等多个模块，为桥牌爱好者提供便捷的计分体验。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring Boot
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
